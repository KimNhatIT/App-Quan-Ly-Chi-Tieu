# App Quản Lý Chi Tiêu

Tính năng chính
1.Xác thực người dùng
  Đăng nhập
  Đăng ký tài khoản
  Tự động chuyển Splash → trang Home
2.Quản lý chi tiêu
  Thêm giao dịch chi tiêu mới
  Hiển thị danh sách giao dịch
  Quản lý danh mục chi tiêu
  Hiển thị chi tiết từng giao dịch
3.Quản lý tài khoản
  Trang Profile
  Trang đổi mật khẩu
  Lưu thông tin tài khoản
4. Phân tích – Thống kê
  Biểu đồ PieChart thể hiện cơ cấu chi tiêu
  Tính tổng các khoản chi theo danh mục
